扩展规范

{
  "id": "google-search",
  "name": "谷歌",
  "version": "0.1.0",
  "category": "搜索",
  "keywords": ["谷歌", "guge", "google", "gg"],
  "openTarget": "https://www.google.com",
  "queryPrefixes": ["谷歌", "guge", "google", "gg"],
  "queryTargetTemplate": "https://www.google.com/search?q={query}"
}

当前最低要求是独立目录中的 `manifest.json`。简单命令可使用 `openTarget`，参数化命令建议支持 `queryPrefixes` 与 `queryTargetTemplate`。
