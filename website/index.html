能力

简单命令和复杂扩展，都走同一套入口。

启动器体验

全局热键呼出、键盘切换、回车执行、右键更多操作。

单文件扩展

普通用户只写一个 JSON 文件，就能增加自己的命令。

参数化命令

支持“前缀 + 参数”，例如 `谷歌 今天的新闻`。

Cloudflare 云同步

扩展元数据、用户安装记录和扩展包都可跨设备同步。

扩展规范

先从单文件 JSON 开始。

当前版本先支持一个扩展目录内的 `manifest.json`。简单指令直接用 JSON 即可，后续再扩展到压缩包和复杂项目结构。 

{
  "id": "google-search",
  "name": "谷歌",
  "version": "0.1.0",
  "category": "搜索",
  "keywords": ["谷歌", "guge", "google", "gg"],
  "queryPrefixes": ["谷歌", "guge", "google", "gg"],
  "queryTargetTemplate": "https://www.google.com/search?q={query}"
}
