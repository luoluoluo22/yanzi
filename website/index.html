产品能力

不是一个单点工具，
而是一套统一的桌面动作系统。

主界面负责搜索与执行

中文、英文、拼音缩写、搜索前缀、脚本传参都走同一个命令面板。键盘党可以直接输入、回车执行，右键继续管理扩展。

鼠标面板负责高频动作

鼠标呼出面板承载高频收藏扩展，支持分组、复制、剪切、粘贴布局，方便快速整理常用工具位置。

扩展以 JSON 为主

打开类、搜索类、脚本类、工作区类扩展统一走单文件 manifest。宿主内置生成提示词、测试执行、错误复制和剪贴板导入。

同步和存储是内建能力

脚本与工作区扩展可以接入本地存储、WebDAV / 坚果云和账户同步，不再需要每个扩展单独重造存储与分发逻辑。

扩展开发

让 AI 生成，
再由宿主真正运行。

燕子的扩展优先走单文件 JSON。搜索、打开、脚本、工作区都用同一份 manifest 描述，再由宿主负责测试、保存、运行和同步。 

{
  "id": "text-length-counter",
  "name": "文本长度统计",
  "queryPrefixes": ["统计", "count"],
  "runtime": "csharp",
  "entryMode": "inline"
}
扩展规范文档 了解 manifest 字段、运行时入口、图标、存储和宿主视图能力。 扩展开发指南 用可复制示例快速写出打开目录、网页搜索、前缀脚本和 hostedViewXaml 扩展。

界面预览

搜索、布局、预览都在同一套桌面语义里。
