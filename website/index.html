设计哲学

不仅仅是启动器， 更是你的数字中枢。

命令优先

无需在层层菜单中寻找。通过关键词和智能参数，直接呼出你想执行的操作，直达终点。

扩展生态

主程序与扩展彻底分离。每一个具体的业务逻辑都由可拔插、可定义的扩展来提供生命力。

极简开发

低门槛是我们的核心。普通用户只需编写一个标准的 JSON 文件，即可将高频操作转化为超级指令。

原生云同步

基于 Cloudflare 全球边缘节点。你的扩展、脚本和配置逻辑始终随你而行，跨越设备边界。

开发生态

不仅仅是启动器。

通过参数化模板，你可以将任何 Web 服务或本地脚本转化为 Swallow 执行流。 

{
  "id": "workflow-ai",
  "name": "燕子助手",
  "version": "1.0.0",
  "keywords": ["ask", "ai"],
  "queryTargetTemplate": "https://chat.me/q={query}"
}
扩展规范文档 深入了解 Swallow 的 manifest 设计。 开发起步 从零开始构建你的第一个 Swallow 插件。
