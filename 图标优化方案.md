# 燕子应用图标优化方案

## 设计原则
1. **统一风格**：所有图标采用 Material Design 风格
2. **视觉平衡**：图标大小和粗细保持一致
3. **清晰易识别**：在小尺寸下也能清晰辨认
4. **颜色规范**：
   - 普通图标：`#A0FFFFFF` (半透明白色)
   - 危险操作：`#EF4444` (红色)
   - 强调图标：`#FBBF24` (金色)
   - 主要图标：`#FFFFFF` (纯白色)

## 实施建议

1. **统一尺寸**：所有图标使用 24x24 的画布
2. **统一粗细**：线条粗细保持一致（约 2px）
3. **统一间距**：图标与边缘保持适当间距
4. **颜色使用**：
   - 默认操作：半透明白色 `#A0FFFFFF`
   - 危险操作：红色 `#EF4444`
   - 特殊标记：金色 `#FBBF24`
   - 强调元素：蓝色 `#60A5FA`

1. `src/OpenQuickHost/MainWindow.xaml` - 主窗口图标
