# 使用说明

## 安装与启动

1. 下载 Windows 压缩包
2. 解压后运行 `OpenQuickHost.exe`
3. 使用 `Ctrl+Shift+Space` 呼出启动器

## 基本操作

- 输入关键字搜索命令
- `Up / Down` 切换条目
- `Enter` 执行命令
- `Esc` 收起启动器
- 右键条目打开更多操作

## 添加扩展

### 方法一：启动器底部 `+`

点击底部状态栏右侧 `+`，粘贴 JSON 内容后保存。

### 方法二：设置页管理

在设置页中后续会继续补完整扩展管理入口。

## 参数化命令示例

- `谷歌 今天的新闻`
- `gg cloudflare pages deploy`
- `guge openai api`

## 云同步

1. 在设置窗口登录账号
2. 本地扩展会在刷新时自动同步
3. 其他设备登录后可以拉取相同扩展记录
