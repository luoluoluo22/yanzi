# 产品说明

## 产品定位

这是一个以“呼出即用、命令优先、扩展可同步”为核心思路的桌面启动器。

目标方向：

1. 主程序和扩展彻底分离
2. 普通用户也能低门槛创建自己的扩展
3. 扩展可以在多设备之间云同步
4. 简单需求用单文件 JSON，复杂需求再扩展到压缩包或更完整的项目结构

## 当前能力

- 全局热键 `Ctrl+Shift+Space` 呼出
- 搜索、筛选、键盘选择、回车执行
- 内置命令与本地扩展统一展示
- 单文件 JSON 扩展新增、编辑、删除
- 条目右键管理：桌面快捷方式、重命名、删除
- Cloudflare 用户登录、扩展元数据同步、扩展包上传下载
- 参数化命令：支持“命令前缀 + 参数”，例如 `谷歌 今天的新闻`

## 设计原则

### 1. 命令优先

用户输入时，应优先命中可执行动作，而不是先进入复杂界面。

### 2. 扩展优先

宿主只负责：

- 命令索引
- 执行入口
- 设置与同步
- 扩展管理

真正的业务动作由扩展提供。

### 3. 低门槛优先

简单扩展应允许用户只写一个 `manifest.json` 就能完成。

### 4. 云同步优先

用户自己的扩展、安装记录和基础配置应能跨设备同步。

## 后续方向

- 支持更多参数化动作类型，如 `url`、`shell`、`http`
- 支持压缩包扩展和复杂项目扩展
- 扩展市场与版本发布
- 自定义热键与主题
- 更完善的扩展权限与签名模型
